# Словарь

| Термин | Значение |
| --- | --- |
| **Хук** | Первые слова, кадр или слайд, которые удерживают внимание и обещают понятное продолжение. |
| **CTA** | Конкретное действие в финале: написать слово, открыть ссылку, сохранить, ответить, перейти к следующему шагу. |
| **VOICE-PROFILE** | Переносимый профиль наблюдаемых правил авторского голоса. |
| **AUDIENCE-INTEL** | Структурированный отчёт по языку, вопросам и паттернам аудитории. |
| **Разрыв любопытства** | Открытый вопрос или факт, который материал обязан честно раскрыть дальше. |
| **Паттерн** | Повторяемая формулировка или реакция, которая встречается у разных людей. |
| **Переупаковка** | Пересборка одной идеи под другую структуру и механику площадки. |
| **Монтажная склейка** | Момент перехода в коротком видео. Для него полезно держать одну законченную мысль. |
| **Регистр** | Допустимая степень формальности, дистанции и разговорности в тексте. |
