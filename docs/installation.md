# Установка и совместимость

## Формат пакета

Каждая папка навыка содержит `SKILL.md` с YAML frontmatter и инструкциями. Устанавливайте папку целиком. Переименование папки допустимо, если имя остаётся понятным и не конфликтует с другим навыком.

## Claude.ai

Claude.ai загружает навыки через интерфейс. Архивируйте одну папку навыка за раз.

```text
hook-lab/
└── SKILL.md
```

После загрузки проверьте простой запрос: `придумай хук для Reels про ...`. Если навык не срабатывает, назовите задачу ближе к его описанию и убедитесь, что тема указана явно.

## Claude Code

Проектный путь:

```text
.claude/skills/<skill-name>/SKILL.md
```

Личный путь:

```text
~/.claude/skills/<skill-name>/SKILL.md
```

Пример:

```bash
mkdir -p .claude/skills
cp -R voice-print .claude/skills/voice-print
```

## Codex

Проектный путь:

```text
.codex/skills/<skill-name>/SKILL.md
```

Личный путь:

```text
~/.codex/skills/<skill-name>/SKILL.md
```

Пример:

```bash
mkdir -p ~/.codex/skills
cp -R audience-dig ~/.codex/skills/audience-dig
```

## OpenCode

OpenCode обнаруживает навыки в нескольких совместимых местах.

```text
.opencode/skills/<skill-name>/SKILL.md
~/.config/opencode/skills/<skill-name>/SKILL.md
.claude/skills/<skill-name>/SKILL.md
~/.claude/skills/<skill-name>/SKILL.md
.agents/skills/<skill-name>/SKILL.md
~/.agents/skills/<skill-name>/SKILL.md
```

Для совместной работы нескольких агентов удобен путь `.agents/skills/`. Для одного проекта подойдёт `.opencode/skills/`.

## Проверка после установки

1. Откройте новую сессию или обновите контекст агента.
2. Дайте один короткий вход, который точно соответствует задаче навыка.
3. Проверьте форму ответа: `VOICE-PROFILE`, `AUDIENCE-INTEL`, таймированный сценарий или разбивка по форматам.
4. При необходимости в явном виде назовите навык: `используй voice-print`.

## Частые проблемы

### Навык не запускается

Проверьте расположение `SKILL.md`, название папки и конкретность запроса. Фраза «помоги с контентом» слишком широка. Фраза «разбери 30 комментариев и найди повторяющиеся вопросы» подходит `audience-dig`.

### Результат звучит слишком обобщённо

Добавьте контекст: площадку, целевую длительность, примеры, факты, фразы аудитории, исходный `VOICE-PROFILE`.

### Мало данных для вывода

Навыки не должны скрывать ограничение. При двух текстах голос предварительный. При восьми комментариях паттерн может быть совпадением. Добавьте материал или используйте результат как гипотезу.
