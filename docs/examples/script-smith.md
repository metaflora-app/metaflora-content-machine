# Пример `script-smith`

## Вход

```text
разверни этот хук в Reels на 45 секунд:
«идеи для контента заканчиваются, когда у тебя нет очереди вопросов от аудитории».
покажи два шага и закончи CTA «сохрани, чтобы разобрать комментарии завтра».
```

## Что проверить в ответе

- есть тайминг блоков
- тело содержит два шага, не пять
- CTA связан с темой и площадкой
- текст можно произнести одним дыханием без книжных оборотов
