# Пример `audience-dig`

## Вход

```text
ниже комментарии под четырьмя роликами. убери личные данные,
найди повторяющиеся вопросы и подготовь три хука на их языке.
[реплики]
```

## Что проверить в ответе

- назван объём материала
- шум не смешан с содержательными репликами
- у паттернов указана частота или степень уверенности
- готовые хуки связаны с реальными словами аудитории
