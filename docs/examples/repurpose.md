# Пример `repurpose`

## Вход

```text
вот сценарий Reels на 30 секунд. пересобери его в карусель на 6 слайдов,
пост в Telegram и письмо. сохрани голос, но не копируй реплики из ролика.
```

## Что проверить в ответе

- карусель разбита на один тезис на слайд
- Telegram-пост читается без визуала
- у письма есть тема и более медленное развитие мысли
- CTA приспособлен под каждый формат
