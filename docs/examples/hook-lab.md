# Пример `hook-lab`

## Вход

```text
сделай варианты первого слайда для карусели про хаос в задачах у фрилансера.
используй pain mirror, curiosity gap и before/after.
без статистики и без кликбейта.
```

## Что проверить в ответе

- каждый вариант подписан формулой
- у каждого есть короткое объяснение
- формулировка помещается на один слайд
- curiosity gap содержит реальный ответ в последующей карусели
