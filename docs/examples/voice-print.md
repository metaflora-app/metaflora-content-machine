# Пример `voice-print`

## Вход

```text
вот шесть моих постов. сделай профиль голоса для будущих сценариев Reels.
[тексты]
```

## Что проверить в ответе

- есть уровень уверенности
- наблюдения привязаны к образцам
- отдельно описаны лексика, ритм, регистр, структура и запреты
- в конце есть одна тестовая фраза

## Как использовать дальше

```text
вот VOICE-PROFILE. напиши сценарий по нему, но не переноси факты и тему из прошлых постов.
```
